My Skills

JavaScript

JavaScript is a programming language. Along with HTML and CSS, JavaScript is one of the three main technologies on the World Wide Web.

CSS3

CSS3 stands for Cascading Style Sheet level 3, which is the advanced version of CSS. It is used for structuring, styling, and formatting web pages.

HTML5

The Hypertext Markup Language revision 5 is a markup language that is used for the structure and presentation of World Wide Web content.

React.js

React.js is a free, open-source JavaScript library. It works best to build user interfaces by combining sections of code (components) into full websites.

Next.js

Next.js is a React.js framework that gives you building blocks to create web applications. One of the main benefits is that it enables server-side rendering.

Bootstrap

It is a free framework for easier web development. It includes HTML and CSS-based design templates for forms, buttons, and many other JavaScript plugins.

Git

Git is a version control system used for tracking changes in computer files. It is generally used for source code management in software development.

GitHub

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.
